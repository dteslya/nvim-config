# My neovim config

This configuration is based on [MiniMax](https://github.com/nvim-mini/MiniMax).

## Usage

- Make a backup of your current Neovim files:

  ```sh
  # required
  mv ~/.config/nvim{,.bak}

  # optional but recommended
  mv ~/.local/share/nvim{,.bak}
  mv ~/.local/state/nvim{,.bak}
  mv ~/.cache/nvim{,.bak}
  ```

- Clone this repo

  ```sh
  git clone https://github.com/dteslya/nvim-config ~/.config/nvim
  ```

- Start Neovim!

  ```sh
  nvim
  ```
